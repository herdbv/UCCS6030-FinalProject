# UCCS6030-FinalProject
Contains the project report, project slides, software artifacts used, and the prompts to used on ChatGPT and Copilot. This repository tested the market-grade LLMs on their ability to generate pre- and post-conditions for basic data structures. 

The data structure tested was a simple linked list with a cursor for position tracking.

The Python-Code directory contains the software artifacts used to test ChatGPT and Copilot.

The Report directory contains the written project report, notes taken, and slideshow used to present the data.
